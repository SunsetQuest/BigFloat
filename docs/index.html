Overview

BigFloat is a library designed to perform high-precision floating point arithmetic. It enables developers to work with numbers beyond the typical precision limits, making it ideal for scientific computations, financial calculations, and more.

Features

  • Arbitrary-precision arithmetic
  • Easy integration with existing projects
  • Robust error handling
  • Optimized performance for high-precision calculations

Installation & Usage

To install BigFloat, simply clone the repository:

git clone https://github.com/SunsetQuest/BigFloat.git

Then include it in your project as needed. Check the documentation for more details on how to get started.

Examples

Below is a simple example of how to use BigFloat in your project:


// Example usage of BigFloat
import { BigFloat } from 'bigfloat';

const a = new BigFloat('123456789.987654321');
const b = new BigFloat('0.00000000123456789');
const result = a.add(b);
console.log(result.toString());
